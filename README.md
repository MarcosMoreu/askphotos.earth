**askphotos.earth** is an experimental initiative in its early stages of development.

* **Issue Reporting:** If you encounter any problems with the **Windows, macOS, or Linux** releases of the GTD Generator, please report them via the [GitHub Issues](https://github.com/MarcosMoreu/GTDGenerator-askphotosdotearth/issues) tab.
* **Workaround:** In the meantime, you can use the more stable CLI approach. You can find the step-by-step guide here: [CLI Instructions Guide](https://github.com/MarcosMoreu/GTDGenerator-askphotosdotearth/blob/main/CLI%20instructions.md).
* **Collaboration & Feedback:** If you would like to get involved, please get in touch using the [online contact form](https://askphotos.earth/pages/contact.html) or via email at [marcos.moreu@ucl.ac.uk](mailto:marcos.moreu@ucl.ac.uk).

🔒 Privacy & Data Security. The technology and workflow are designed so that askphotos.earth does not have access to any of your/the crowd data. 

⚖️ Disclaimer. The askphotos.earth initiative has made every effort to develop a tool that converts Google Photos albums into maps with the highest possible accuracy. However, we cannot accept responsibility for any errors, omissions, or inconsistencies that may occur in the generated data.
